Figures

Figures are handled in one of two ways. The first way is to use Pandoc’s shorthand:

Figure 1: This is a Pandoc figure.

I also sometimes use description lists with class figure to display figures. The dd element holds the image and captions are contained inside <dt><p> ... </p></dt> blocks.

Figure 1 (alt): This is a description list figure.

The second option allows more flexibility since the images can be made into links, and multiple images can be added to the same figure.

No Stretching

By default, figures are stretched to 95% of the text width. This can be suppressed by surrounding the figure in a div with class nostretch:

Figure 1: This is a Pandoc figure.
